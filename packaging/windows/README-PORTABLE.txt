chatxz — Windows portable
=========================

No installer. Unzip anywhere and run.

Quick start
-----------
1. Unzip this folder to a location you control (Desktop, Downloads, D:\apps, etc.).
2. Double-click chatxz.exe
3. Your browser opens to http://127.0.0.1:8742
4. If Windows Firewall asks, allow chatxz on private networks (UDP 4242 + TCP 8742).

Stop chatxz
-----------
Close the black console window, or press Ctrl+C in that window.

Your data
---------
Everything stays in this folder:
  chatxz-data\     identity, settings, chat history, received files

LAN access
----------
Other devices on your Wi‑Fi/LAN can open:
  http://<your-pc-ip>:8742
Find your IP: open Command Prompt and run  ipconfig

Troubleshooting
---------------
- "Port already in use": close any other chatxz window, or reboot.
- Antivirus blocked the app: add an exclusion for this folder.
- Peers not found: click Announce in the web UI; ensure both PCs are on the same LAN.
